# Schwab API Research Report (Finalized Findings)
1. Schwab API Data Availability
Finding
Schwab provides market data through the Market Data API. The API allows applications to retrieve financial market information for supported securities.
The project will use Schwab only as a data provider.
The application will:
Retrieve market data
Store historical records locally
Provide access through a custom API
The application will not:
Execute trades
Modify user accounts
Manage portfolios

3. Historical Data Limitations
Research Finding
Schwab does not provide unlimited historical data.
Approximate limitations:

Up to approximately 15-20 years depending on security

Architectural Decision
Because the MVP focuses on daily historical data:
The limitation is not a blocker.

10. Final MVP Success Criteria
The MVP is complete when:
A user can create a watchlist, the system automatically retrieves daily historical market data from Schwab after market close, stores the information in a database, and allows users to retrieve and visualize the data through an API and web interface.
